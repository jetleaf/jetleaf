CommandLineRunner

CommandLineRunner class abstract interface

A runner that receives raw command-line arguments.

CommandLineRunner is typically used when you want to process arguments exactly as passed to the process (i.e. the main(List<String> args) signature in Dart).

Example:

final class MyRunner implements CommandLineRunner {
  @override
  void run(List<String> args) {
    print('Raw args: $args');
  }
}
