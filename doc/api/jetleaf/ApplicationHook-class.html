ApplicationHook

ApplicationHook class abstract interface

A hook interface for integrating with the lifecycle of a Application.

Implementations of this interface can register listeners that observe and react to application startup events such as context preparation, environment loading, and shutdown.

This is useful for frameworks or extensions that want to plug into the application bootstrap process without modifying the application code.

Example:

class LoggingHook implements ApplicationHook {
  @override
  ApplicationRunListener getRunListener(Application app) {
    return LoggingRunListener(app);
  }
}
