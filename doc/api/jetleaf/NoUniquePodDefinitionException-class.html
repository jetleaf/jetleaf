NoUniquePodDefinitionException

NoUniquePodDefinitionException class

Exception thrown when multiple pod definitions are found for a given type in the application context.

This exception typically occurs when trying to retrieve a pod by its type from the application context, but multiple matching pod definitions are found, making it ambiguous which pod to use.

Example

throw NoUniquePodDefinitionException('Multiple pod definitions found for type MyService');
