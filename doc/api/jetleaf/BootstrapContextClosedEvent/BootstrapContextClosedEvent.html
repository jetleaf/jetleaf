An ApplicationEvent published when the BootstrapContext is closed, typically just before or during the early phase of starting a full ConfigurableApplicationContext.

This event signals that the bootstrap phase is complete and allows interested listeners to perform cleanup, finalize preparation, or transition to the full application context.

Example:

class MyListener implements ApplicationListener<BootstrapContextClosedEvent> {
  @override
  void onApplicationEvent(BootstrapContextClosedEvent event) {
    final bootstrap = event.getBootstrapContext();
    final appContext = event.getApplicationContext();
    print("Bootstrap context closed, switching to: $appContext");
  }
}

Creates a new event that indicates the source bootstrap context has been closed and applicationContext is now in use.
