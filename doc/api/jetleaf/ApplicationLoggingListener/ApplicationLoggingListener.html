JetLeaf's application-level LoggingListener implementation.

This listener acts as a bridge between the JetLeaf logging system and the active Environment. It configures a Logger instance per application or per log tag, using environment-driven settings from JetLoggingProperty.

Features:

  • Creates a lazily-initialized Logger per tag.
  • Applies formatting and output options (pretty, flat, json, etc.).
  • Supports both global (logging.level.all) and tag-specific (logging.level.<tag>) log levels.
  • Filters log messages before delegating to the Logger.

Example:

final listener = ApplicationLoggingListener(MyApp, environment);
LoggingSystem.addListener(listener);

Creates an ApplicationLoggingListener bound to the given _applicationClass and _environment.
